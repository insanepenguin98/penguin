html문서만들기
-------------------------
html 문서 만들기 주의 사항
1. 웹 문서 저장 시 반드시 영문 소문자로 저장해야 함.
2. 웹 문서의 이름을 정할 때 숫자가 먼저 나올 수 없음.
3. 웹 문서의 이름을 정할 때 한글사용 할 수 없음.
4. 웹 문서의 이름 정할때 공백없이 쓰고 공백이 필요할 경우 _(언더바), -(하이픈)으로 작성함.

웹 표준이란?
================
w3c(world wide web : 웹표준 국제 기구)에서 정의한 태그로 웹 구조를 만드는것.
개발자들간의 약속으로 일관된 코드를 구현하여 "어떤 브라우저를 이용하던지 같은 화면을
볼수 있게 하고(=크로스 브라우징)", 시각장애인들에게 본문과 본문외의 내용을 쉽고 빠르게 파악할 수 있게 해주고,
플러그인 설치를 통한 악성바이러스에서 보호될 수 있게 만들어 주는것이 웹표준.
